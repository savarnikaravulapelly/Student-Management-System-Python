# 🎓 Student Management System (Python)

This is a console-based Python application developed to manage student records efficiently using file handling.

## 🚀 Features

- Add new student records
- View all students
- Search student by roll number
- Persistent data storage using text files

## 🛠 Technologies Used

- Python
- File Handling
- OS Module

## 📂 Project Structure

Student-Management-System-Python/
│── student_manager.py
│── students.txt (auto-generated)
